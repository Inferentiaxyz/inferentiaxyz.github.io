Guida per installare un LLM sul tuo PC!

  • Apr 18, 2023

Il modello Vicuna è considerato al momento il miglior chatbot gratuito. Vicuna-13B raggiunge una qualità del 90%* rispetto a OpenAI ChatGPT e Google Bard, superando altri modelli come LLaMA e Stanford Alpaca in oltre il 90%* dei casi.

In questo articolo, ti guideremo su come eseguire il modello Vicuna sul tuo computer locale, utilizzando la tua CPU o GPU.

Conda

Il primo step da effettuare, sia se si vuole usare CPU che GPU è quello di installare Conda. Conda è un gestore di pacchetti open-source che viene utilizzato principalmente per la gestione degli ambienti di sviluppo in Python. Conda permette di creare facilmente ambienti virtuali isolati e gestire le dipendenze dei pacchetti in modo efficiente, permettendo così di lavorare su progetti con diverse versioni di librerie e pacchetti senza incorrere in problemi di compatibilità.

wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh
sha256sum Miniconda3-latest-Linux-x86_64.sh
bash Miniconda3-latest-Linux-x86_64.sh
source ~/.bashrc

Installazione per GPU (GPTQ Quantised)

Per installare il modello Vicuna con la tua GPU, devi prima creare un ambiente virtuale utilizzando Conda.

conda create -n vicuna python=3.9
conda activate vicuna

Successivamente, installa l'interfaccia web per interagire con il modello Vicuna. 

git clone https://github.com/oobabooga/text-generation-webui.git
cd text-generation-webui
pip install -r requirements.txt

Per utilizzare la versione del modello quantizzato GPTQ scarichiamo la seguente repository in una sottocartella.

mkdir repositories
cd repositories
git clone https://github.com/oobabooga/GPTQ-for-LLaMa.git -b cuda
cd GPTQ-for-LLaMa
python setup_cuda.py install

Adesso possiamo scaricare il modello

cd ../..
python download-model.py anon8231489123/vicuna-13b-GPTQ-4bit-128g

Infine possiamo quindi avviare l'interfaccia web e utilizzare il modello Vicuna.

python server.py --chat --model anon8231489123_vicuna-13b-GPTQ-4bit-128g --model_type LLaMA --wbits 4 --groupsize 128 --share

Installazione per CPU (GGML Quantised)

Iniziamo con il creare un ambiente virtuale utilizzando Conda.

conda create -n vicuna python=3.9
conda activate vicuna

In seguito, cloniamo ed installiamo il repository llama.cpp

  git clone https://github.com/ggerganov/llama.cpp.git
  cd llama.cpp
  make

Per scaricare i binari del modello quantizzato possiamo utilizzare il modulo Python huggingface-hub che si installa con il seguente codice:

pip install huggingface-hub

Adesso utilizzando l'interprete python possiamo scaricare il modello

python

>>> from huggingface_hub import hf_hub_download
>>> hf_hub_download(repo_id="eachadea/ggml-vicuna-13b-4bit", filename="ggml-vicuna-13b-4bit.bin", local_dir="./models")
>>> exit()

Infine per interagire con il modello eseguiamo:

  ./main -m ./models/ggml-vicuna-13b-4bit.bin --color -f ./prompts/alpaca.txt -ins -b 256 --top_k 10000 --temp 0.2 --repeat_penalty 1 -t 7
